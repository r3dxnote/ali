## ⚠️ إخلاء مسؤولية هام (FIFA Disclaimer)
تطبيق **"كورة علي" (Ali World Cup Match Center)** هو تطبيق ويب تجريبي (Demo/PWA Sandbox) مخصص للعرض الفني والتحقق من كفاءة تطبيقات الويب التقدمية. **جميع البيانات المعروضة (نتائج، جداول، إحصائيات) هي بيانات محاكاة تجريبية وليست حقيقية، وليس للموقع أي صلة أو تبعية رسمية بالاتحاد الدولي لكرة القدم (FIFA) أو أي جهة تنظيمية رسمية للبطولة**.

## ⚙️ مميزات النسخة V1-C (PWA & Branding)

1.  **هوية بصرية جديدة:** تم توحيد اسم التطبيق عبر الواجهات والبيانات الوصفية باسم **"كورة علي"** باللغة العربية، و **"Ali World Cup Match Center"** باللغة الإنجليزية.
2.  **تطبيق الويب التقدمي (PWA):**
    *   إدراج ملف تبيان الويب `manifest.webmanifest` متوافق تماماً مع المسارات النسبية لـ GitHub Pages.
    *   إضافة رموز الهواتف المحمولة وتخصيص ألوان شريط الحالة والثيم المونديالي العنابي (`#8a1538`).
    *   التثبيت المباشر على الشاشة الرئيسية للهواتف الذكية (iOS & Android) كـ تطبيق مستقل (Standalone).
3.  **رموز التطبيق الفاخرة (PNG & SVG Icons):**
    *   تم توليد رموز PNG عالية الدقة بأحجام `192x192` و `512x512` ومقاس قناع آمن `maskable` للاندرويد، بالإضافة لرمز متجهة SVG كأصل إضافي مرن.
4.  **عامل الخدمة المحافظ (Service Worker Caching):**
    *   تم تأسيس عامل الخدمة `service-worker.js` لتسجيل وحفظ ملفات الهيكل البرمجي (App Shell) وملفات البيانات المحلية في ذاكرة الكاش `ali-wc-v1c` لتمكين التصفح الفوري في حالة عدم وجود شبكة (Offline Mode).
    *   يقوم تلقائياً بتنظيف مخازن الكاش القديمة التي تبدأ بـ `ali-wc-` عند التحديث لضمان وصول التحديثات للمستخدم فورياً دون تعليق التطبيق.

## 🚀 كيفية التشغيل والاختبار محلياً

بسبب سياسات الأمان في المتصفحات الخاصة بـ CORS، يجب استخدام خادم محلي لتشغيل التطبيق والتحقق من تسجيل عامل الخدمة بنجاح:

```bash
python -m http.server 8080
```

افتح المتصفح على: [http://localhost:8080](http://localhost:8080)
ثم تفقد علامة تبويب "Application -> Service Workers" في أدوات المطورين (F12) لمراقبة نشاط الكاش وعامل الخدمة.
