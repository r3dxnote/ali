# كُورة مُوند - مركز مباريات كأس العالم 2026 (النسخة V1)

موقع ويب تجريبي متجاوب بالكامل مع الهواتف (Mobile-First) لجدول ومجموعات مباريات كأس العالم 2026، مبني باستخدام التقنيات الأساسية النظيفة (Vanilla HTML, CSS, JS) ودون أي متطلبات أو مكتبات خارجية.

رابط الموقع التجريبي على صفحات جيتهاب:
[https://r3dxnote.github.io/ali/](https://r3dxnote.github.io/ali/)

---

## ⚠️ تنبيه هام (Disclaimer / Demo Data)
جميع البيانات المعروضة في هذا الموقع (أوقات، ملاعب، نتائج، تصنيفات المجموعات) هي **بيانات تجريبية ومحاكاة فقط (Demo/Sample Data)** مخصصة لغرض التحقق التقني واختبار واجهة المستخدم لنسخة V1. **لا يمثل هذا الموقع أي جهة رسمية وليست البيانات حية أو دقيقة**.

---

## 🛠️ هيكلية الملفات

*   `index.html`: هيكل التطبيق والصفحة الرئيسية الداعمة للكتابة من اليمين لليسار (RTL).
*   `.nojekyll`: ملف إرشادي لإخبار خوادم GitHub Pages بعدم تشغيل معالج Jekyll على ملفات الأصول.
*   `assets/css/app.css`: ملف التصميم والألوان المونديالية المتناسقة المعتمدة على خطوط النظام العربية والـ CSS Grid/Flex.
*   `assets/js/app.js`: كود الاستعلام والجلب (`fetch()`) لملفات JSON، والتحكم بالتبويبات الفورية وصياغة الوقت العربي.
*   `assets/data/meta.json`: إعدادات الموقع وتاريخ اليوم التجريبي (`demoToday: 2026-06-17`).
*   `assets/data/matches.json`: مصفوفة بيانات المباريات والدور الأول وجدول ترتيب المجموعات.

## 🚀 كيفية التشغيل والاختبار محلياً

بسبب قيود الحماية في متصفحات الويب الحديثة الخاصة بـ CORS عند جلب ملفات JSON محلياً عن طريق مسار الملف المباشر (`file://`)، **يُوصى بتشغيل خادم محلي** لاختبار الموقع.

يمكنك تشغيل خادم محلي بسيط باستخدام بايثون في موجه الأوامر داخل مجلد المشروع:

```bash
python -m http.server 8080
```

ثم افتح المتصفح على العنوان التالي:
[http://localhost:8080](http://localhost:8080)
